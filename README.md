# steem-sync

> 스팀잇 컨텐츠 로컬 동기화를 목표로 함. 빠른 검색을 위함

## 설치 및 실행

```sh
# `degit` 또는 `git clone` 을 통해 저장소(repository) 복제
degit https://github.com/wonsama/steem-sync

# `의존성 설치`
yarn

# `실행`
yarn dev
```
